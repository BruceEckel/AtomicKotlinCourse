## Summary 1 (#2)

Reverse a decimal number using a while loop. The auxiliary `worker` variable
can help you find the `result` iteratively:

| Worker     | Result |
|------------|--------|
| 1234       | 0      |
| 123        | 4      |
| 12         | 43     |
| 1          | 432    |
| 0          | 4321   |
